Code à utiliser pour le protocole TCP de mon serveur écrit en héxadécimale :
4103 {...}(0x1007) : insertion d'un produit
8201 (0x2009) : voir le nombre de clients
12288 (0x3000) : lance le test de latence du serveur, il répond trop lentement le programme client se termine
12289 (0x3001) : essayer de faire un dépassement de buffer au serveur
12290 (0x3002) : message par défaut envoyé toutes les 5sec
61440 (0xF000) : quitter l'application
